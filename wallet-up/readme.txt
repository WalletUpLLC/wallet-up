== Description ==

The Wallet Up "Virtual Wallet" has it all: Easily Integrate Zelle, Facebook pay, Cash App, Venmo, PayPal with a generated QR Code anywhere on your Online Web Site. Get started now!


== How to install? ==

- Go to Plugins -> Add New
- Click Upload Plugin
- Click Choose File to Locate the ‘wallet-up.zip’
- Click on Install Now
- Activate the plugin


= What's next? =

After activation go to Wallet Up settings to set up your accounts profiles. Visit https://walletup.app/wallet-up for a demo.

= What do we support? =

Wallet Up plugin currently supports Cash App, Venmo, and PayPal. Zelle, Facebook Pay and more are coming soon.
